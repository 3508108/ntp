# ntp
Актуальний робочий проєкт у цьому репозиторії — `easy/`.

Раніше в корені була окрема legacy-кодова база `ntp-dashboard`; її прибрано, щоб уникнути дублювання й плутанини.

## Що зараз підтримується
- `easy/` — Go-сервіс збору NTP-часу (apple/google/nist), SQLite, API + SSE.
- `easy/easy.service` — systemd unit для продакшен-запуску.
- `easy/PROJECT.md` — актуальна операційна документація.
- `easy/AGENTS.md` — правила роботи агентів у цьому підпроєкті.

## Швидкий старт
```bash
cd easy
go build ./...
go run .
```

## Деплой
Використовуйте `easy/Makefile` (systemd-сценарій без Docker):
```bash
cd easy
make deploy
```

Поточна інфраструктура DNS працює на двох Frankfurt-серверах:
- `fra-dns` — `64.226.73.107` / `2a03:b0c0:3:f0:0:2:8fa9:e000`
- `fra-dns-2` — `165.227.157.91` / `2a03:b0c0:3:f0:0:2:8fbc:f000`
